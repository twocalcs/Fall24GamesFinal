void collision::detectColisions ( Scene  sc)

Detect colision in scene.

Recursivly loads all game objects in the top level scene and all subsceens, then uses broad phase collision detection to detect colisions between game objects

When a colision is detected, the OnColision method is called on both game objects, passing the other game object and the intersection rect
